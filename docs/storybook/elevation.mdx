# Elevation
Elevations are layered surfaces that form the foundation of UI.
It provides cues about the surface depth and stacking order of elements in an experience.
They create a blank canvas where other UI will be placed, such as text, icons, backgrounds, and borders.

Elevation tokens are used to give elements the appearance of being raised off the page. \
Use them with the box-shadow property. \
These are especially useful for menus, popovers, and dialogs.

## Shadows

Shadows are used to give the impression of distance or elevation between surfaces, which adds depth to our designs.
