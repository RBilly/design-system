# Typography

## Typefaces

Noto is a joint project by Google, Adobe, and Monotype with extensive language support. \
The Noto name, short for "no tofu," is explained by the project’s mission to eliminate "Tofu", the blank rectangles that look like tofu that are shown when no font is available for your text.
Noto Sans is a global font collection with unmodulated (“sans serif”) design for writing in all modern and ancient languages. \
Noto Sans communicates openness and has a neutral tone. In our user interfaces, we use the following typefaces from this collection:

* [Noto Sans Display](https://fonts.google.com/noto/specimen/Noto+Sans+Display) for texts in the Latin, Cyrillic and Greek scripts.
* [Noto Sans Simplified Chinese](https://fonts.google.com/noto/specimen/Noto+Sans+SC) for languages in mainland China that use the Simplified Chinese variant of the Han ideograms.
* [Noto Sans Japanese](https://fonts.google.com/noto/specimen/Noto+Sans+JP) for the Japanese language and other languages used in Japan.
* [Noto Sans Korean](https://fonts.google.com/noto/specimen/Noto+Sans+KR) for the Korean language using Hangul and the Korean Hanja scripts.

## Usage

If you are looking for a document referencing all typography tokens, please visit the [Dictionary section](../?path=/docs/fundamentals-design-tokens--docs#dictionary) and open up the **Typography** panel.

### SCSS

To get the expected typography token, build the variable name using `$typography` prefix followed by the typography category (`Heading`,`Body`,`Label`) and then the element type. \
For example, the typography of an `H3` element correspond to the `typographyHeadingH3` token. And the typography of an `medium` body element correspond to the `$typographyBodyMedium` token.

<Source dark="true" language="css" code={`
@use '@bio4c/ui-kit/designTokens';

p {
  font: designTokens.$typographyBodyMedium;
}
`}/>

### JavaScript
